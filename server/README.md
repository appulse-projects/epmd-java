### As library

Include the dependency to your project's pom.xml file:

```xml
<dependencies>
    ...
    <dependency>
        <groupId>io.appulse.epmd.java</groupId>
        <artifactId>server</artifactId>
        <version>0.3.2</version>
    </dependency>
    ...
</dependencies>
```

or Gradle:

```groovy
compile 'io.appulse.epmd.java:server:0.3.2'
```
